# <a name="initializing-test-cases-oracletosql"></a>初始化测试用例 (OracleToSQL)
在此步骤中，您将为新的测试用例指定初始信息。  
  
## <a name="parameters"></a>参数  
**测试用例名称**  
输入名称以标识测试用例。  
  
**创建日期**  
今天的当前日期，自动定义。  
  
**上次修改日期**  
自动填充;不应更改。  
  
**测试用例描述**  
输入任何其他信息以确定测试用例的用途。
