# <a name="convert-load-and-migrate-accesstosql"></a>转换、加载和迁移（AccessToSQL）
使用 "转换"、"加载" 和 "迁移" 按钮或迁移向导时，"**转换"、"加载" 和 "迁移**" 对话框将报告迁移状态。  
  
迁移完成后，可以通过单击每个部分中的链接来查看信息性、警告和错误消息。 该链接显示可用的错误、警告和信息性消息的数量，如 "0 个错误，4个警告"。  
  
页面的 "**迁移所选对象的数据**" 部分还包含 "**报表**" 按钮。 您可以单击此按钮以查看数据迁移报表。  
  
## <a name="options"></a>选项  
**报告**  
显示关联的转换或迁移报表。
