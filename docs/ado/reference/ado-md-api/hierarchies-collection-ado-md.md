# <a name="hierarchies-collection-ado-md"></a>层次结构集合 (ADO MD)
包含维度中的 [层次结构](./hierarchy-object-ado-md.md) 对象集。  
  
## <a name="remarks"></a>注解  
 **层次结构** 是标准 ADO 集合。 通过集合的属性和方法，你可以执行以下操作：  
  
-   获取集合中具有 [Count](../ado-api/count-property-ado.md) 属性的对象的数目。  
  
-   返回集合中具有默认 [项](../ado-api/item-property-ado.md) 属性的对象。  
  
-   用 [Refresh](../ado-api/refresh-method-ado.md) 方法更新集合中的对象。
