# <a name="dmx-statements---data-definition"></a>DMX 语句 - 数据定义 
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  下表列出了数据挖掘扩展插件 (DMX) 中属于数据挖掘数据定义语言 (DDL) 的语句。  
  
|声明专用纸|描述|  
|---------------|-----------------|  
|[CREATE MINING STRUCTURE (DMX)](../dmx/create-mining-structure-dmx.md)|在数据库中创建新挖掘结构。|  
|[更改挖掘结构&#40;DMX&#41;](../dmx/alter-mining-structure-dmx.md)|向现有挖掘结构中添加挖掘模型。|  
|[创建挖掘模型&#40;DMX&#41;](../dmx/create-mining-model-dmx.md)|在数据库中创建新挖掘结构和挖掘模型。|  
|[删除挖掘模型 (DMX)](../dmx/drop-mining-model-dmx.md)|从数据库中删除挖掘模型。|  
|[删除挖掘结构&#40;DMX&#41;](../dmx/drop-mining-structure-dmx.md)|从数据库中删除挖掘结构。|  
|[EXPORT (DMX)](../dmx/export-dmx.md)|将挖掘模型或挖掘结构及关联对象导出到 .abf 文件中。|  
|[IMPORT (DMX)](../dmx/import-dmx.md)|从 .abf 文件中导入挖掘模型或挖掘结构及关联对象。|  
|[选择加入&#40;DMX&#41;](../dmx/select-into-dmx.md)|创建现有挖掘模型的副本。|
