# <a name="ready-to-repair"></a>准备修复
  “准备修复”页提供 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 安装程序执行的修复过程的摘要。  
  
## <a name="options"></a>选项  
 此页上有下列选项：  
  
-   若要继续安装，请单击 **“修复”**。  
  
-   若要更改安装设置，请单击 **“上一步”**。  
  
-   若要退出安装程序，请单击 **“取消”**。
