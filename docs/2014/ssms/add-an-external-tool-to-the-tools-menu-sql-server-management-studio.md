# <a name="add-an-external-tool-to-the-tools-menu-sql-server-management-studio"></a>在“工具”菜单中添加外部工具 (SQL Server Management Studio)
  可以从 [!INCLUDE[msCoName](../includes/msconame-md.md)] 启动任何 [!INCLUDE[msCoName](../includes/msconame-md.md)] Windows 或 [!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)] [!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)] 应用程序。 可以在“工具”  菜单中添加和运行外部应用程序。  
  
### <a name="to-add-an-external-tool-to-the-tools-menu"></a>在“工具”菜单中添加外部工具  
  
1.  在“工具”  菜单上，单击“外部工具”  。  
  
2.  在“标题”  文本框中，键入要显示在“菜单内容”  列表中的名称。  
  
3.  在“命令”  文本框中，键入程序名。 必要时，可包括指向可执行文件的路径。  
  
4.  必要时，可在“参数”  文本框中键入程序参数。  
  
5.  必要时，可在“初始目录”  文本框中键入程序的初始目录。  
  
6.  若要向“菜单内容”  列表添加工具，请单击“添加”  ，再单击“确定”  。
