# <a name="open-an-existing-solution"></a>打开现有解决方案
  在解决方案资源管理器中一次只能打开一个解决方案。  
  
### <a name="to-open-an-existing-solution"></a>打开现有的解决方案  
  
1.  在“文件”菜单中，指向“打开”，然后单击“项目”/“解决方案”。  
  
2.  在“打开项目”对话框中，定位到所需的解决方案，单击该解决方案，再单击“打开”。
