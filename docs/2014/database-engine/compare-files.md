# <a name="compare-files"></a>比较文件
  可以对文件进行比较，确定文件是如何进展到其现有状态的。 例如，如果在签入特定的源文件版本后，在代码项目的内部版本中检测到一个缺陷，则可以将当前文件版本与以前的版本进行比较。 这有助于精确定位导致缺陷的代码。  
  
 可以使用 [!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)]，将项目或文件的本地副本与存储在源代码管理中的版本进行比较，也可以比较两个本地文件。 此外，使用**历史记录**命令时，您可以比较任意两个受源代码管理版本。  
  
### <a name="to-compare-files"></a>比较文件  
  
1.  在解决方案资源管理器中，选择一个项目或其中一个项目文件。  
  
2.  上**文件**菜单，依次指向**源代码管理**，然后单击**比较**。  
  
3.  在中**差异选项**对话框中，选择相应的选项，，然后单击**报表**。
