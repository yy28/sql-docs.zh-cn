# <a name="choose-the-deployment-location"></a>选择部署位置
  使用报表向导中的此页可为报表服务器项目设置部署服务器和文件夹。  
  
## <a name="options"></a>选项  
 **报表服务器**  
 键入 Report Server 的虚拟目录的 URL，例如， http://localhost/reportserver。 这是报表服务器的虚拟目录，而不是报表管理器的虚拟目录。 默认情况下，报表服务器在安装后使用的虚拟目录名为 reportserver。  
  
 **部署文件夹**  
 键入在发布报表服务器项目中所包含报表时将使用的目标文件夹的名称。 文件夹的默认名称为项目的名称。 如果报表服务器上没有该文件夹，则报表设计器将在发布报表时创建该文件夹。
