# <a name="enter-data-source-credentials-dialog-box-report-builder"></a>“输入数据源凭据”对话框（报表生成器）
  使用 **“输入数据源凭据”** 对话框可以提供您用来从数据源检索数据的凭据。  
  
## <a name="options"></a>选项  
 **用户名**  
 键入登录数据源时要使用的用户名。  
  
 **密码**  
 键入登录数据源时要使用的密码。
