# <a name="report-data-f1-help"></a>报表数据的 F1 帮助
  本节中的主题提供您可以对数据源、共享数据源和数据集进行设置的选项有关的信息，以及有关在报表设计器中提供的查询设计器的信息。  
  
 若要从 **“报表属性”** 对话框中访问这些主题，请单击 **“帮助”** 。
