# <a name="report-properties-dialog-box-code"></a>“报表属性”对话框 -&gt;“代码”
  在 "**报表属性**" 对话框中选择 "**代码**" 可以定义将由报表中的表达式使用的自定义代码。  
  
## <a name="options"></a>选项  
 **自定义代码**  
 键入要在报表中使用的方法。 必须编写[!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[vbprvb](../includes/vbprvb-md.md)]代码。
