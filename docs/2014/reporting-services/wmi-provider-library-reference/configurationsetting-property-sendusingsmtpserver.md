# <a name="sendusingsmtpserver-property-wmi-msreportserver_configurationsetting"></a>SendUsingSMTPServer 属性 (WMI MSReportServer_ConfigurationSetting)
  指定报表服务器的电子邮件配置中的 `SendUsing` 元素是否设置为 `true`。  
  
## <a name="syntax"></a>语法  
  
```vb  
Public Dim SendUsingSMTPServer As Boolean  
```  
  
```csharp  
public Boolean SendUsingSMTPServer;  
```  
  
## <a name="property-values"></a>属性值  
 只读的 `Boolean` 对象。
