# <a name="iswebserviceenabled-property-wmi-msreportserver_configurationsetting"></a>IsWebServiceEnabled 属性 (WMI MSReportServer_ConfigurationSetting)
  指示是否已启用报表服务器 Web 服务。 只读。  
  
## <a name="syntax"></a>语法  
  
```vb  
Public Dim IsWebServiceEnabled As Boolean  
```  
  
```csharp  
public boolean IsWebServiceEnabled;  
```  
  
## <a name="property-values"></a>属性值  
 只读的 `Boolean` 值。 如果值为 `true`，指示报表服务器 Web 服务已启用。
