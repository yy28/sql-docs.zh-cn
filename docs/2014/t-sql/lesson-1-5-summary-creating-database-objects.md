# <a name="summary-creating-database-objects"></a>摘要：创建数据库对象
  在本教程中，您已创建了一个数据库，在数据库中创建了表，将数据插入到表中，更改了数据，然后又读取了表中的数据。 所用语句的语法仅仅是基本语法，许多语法选项并未在本教程中涉及。 若要了解有关这些语句的详细信息，请阅读 SQL Server 联机丛书中这些语句的完整语法，并查看这些主题中提供的许多示例。
