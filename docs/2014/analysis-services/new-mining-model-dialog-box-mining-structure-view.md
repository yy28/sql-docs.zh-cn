# <a name="new-mining-model-dialog-box-mining-structure-view"></a>“新建挖掘模型”对话框（“挖掘结构”视图）
  可以使用 **“新建挖掘模型”** 对话框在挖掘结构中添加新的挖掘模型。  
  
## <a name="options"></a>选项  
 **模型名称**  
 为新挖掘模型提供唯一的名称。 该名称不仅对于挖掘结构是唯一的，而且对于数据库也必须是唯一的。  
  
 **算法名称**  
 选择新挖掘模型所基于的算法。
