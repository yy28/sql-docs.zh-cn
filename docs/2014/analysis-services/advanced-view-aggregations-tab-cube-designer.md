# <a name="advanced-view-aggregations-tab-cube-designer"></a>高级视图（多维数据集设计器的“聚合”选项卡）
  可以使用聚合设计器的 **“高级视图”** 手动创建和修改聚合设计及单个聚合。 此视图应仅允许高级用户使用。  
  
 **注意**"**聚合设计向导**" 和 "**基于使用情况的优化向导**" 在此视图中不可用。  
  
## <a name="options"></a>选项  
  
|选项|说明|  
|------------|-----------------|  
|**新建聚合**|单击以创建新的聚合。|  
|**删除**|单击以删除选定的聚合或聚合设计。|
