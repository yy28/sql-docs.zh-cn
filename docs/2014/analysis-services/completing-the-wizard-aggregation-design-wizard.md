# <a name="completing-the-wizard-aggregation-design-wizard"></a>完成向导（聚合设计向导）
  可以使用 **“完成向导”** 页将聚合设计保存到选定分区。 您也可以根据需要在聚合设计向导完成设计并保存聚合之后，部署并处理所选的分区。  
  
## <a name="options"></a>选项  
 **名称**  
 输入聚合设计的名称。  
  
 **立即部署并处理聚合**  
 完成该向导，将聚合设计分配到所选分区，然后部署并处理所选分区。  
  
 **保存聚合但不是处理它们**  
 完成该向导，并将聚合设计分配到所选分区，但不部署或处理所选分区。
