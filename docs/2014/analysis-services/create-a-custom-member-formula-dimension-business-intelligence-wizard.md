# <a name="create-a-custom-member-formula-dimension-business-intelligence-wizard"></a>创建自定义成员公式（维度）（商业智能向导）
  使用 "**创建自定义成员公式**" 页可以选择商业智能向导将属性与自定义成员公式列关联的维度-包含定义该属性的成员值的多维表达式（MDX）语句的列。  
  
## <a name="options"></a>选项  
 **维度**  
 选择维度，向导会将此维度中包含的属性与自定义成员公式关联。
