# <a name="specify-a-unary-operator-dimension-business-intelligence-wizard"></a>指定一元运算符（维度）（商业智能向导）
  可以使用 **“指定一元运算符”** 页选择一个维度，该维度具有定义的父属性并且商业智能向导将为其应用一元运算符。 有关父属性的详细信息，请参阅 [属性和属性层次结构](multidimensional-models-olap-logical-dimension-objects/attributes-and-attribute-hierarchies.md)。  
  
> [!NOTE]  
>  如果从维度设计器或者通过在 [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)]的解决方案资源管理器中右键单击维度启动了商业智能向导，则将不会显示此页。  
  
## <a name="options"></a>选项  
 **维度**  
 选择一个维度，该维度包含父属性并且商业智能向导将对其应用一元运算符。
