# <a name="select-table-dialog-box-mining-accuracy-chart-view"></a>“选择表”对话框（“挖掘准确性图表”视图）
  可以使用 **“选择表”** 对话框选择一个表，用于测试选定挖掘结构中挖掘模型的准确性。  
  
## <a name="options"></a>选项  
 **数据源**  
 包含将用于测试挖掘模型的表的数据源。 该列表包括当前 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 项目中定义的每个数据源。  
  
 **表/视图名称**  
 选择当前选定数据源中可用的表或视图。
