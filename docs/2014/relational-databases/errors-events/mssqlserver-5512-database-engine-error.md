## <a name="details"></a>详细信息  
  
|||  
|-|-|  
|产品名称|MSSQLSERVER|  
|事件 ID|5512|  
|事件源|MSSQLSERVER|  
|组件|SQLEngine|  
|符号名称|FS_TRUNCATEDIR_FAILED|  
|消息正文|截断目录“%.*ls”时遇到错误 0x%x (%ls)。|
