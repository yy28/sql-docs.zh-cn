# <a name="results-detailed-view-dialog-box"></a>“结果详细视图”对话框
  使用 **“评估策略”** 对话框运行策略并单击 **“评估”** 后，此对话框显示策略评估结果。 此对话框是只读的，可帮助您了解属性表达式的哪个部分可能失败。  
  
## <a name="options"></a>选项  
 **AndOr**  
 当存在多个属性表达式时，指示属性表达式是累积还是替代性的。  
  
 **结果**  
 表示属性表达式成功或失败的图标。  
  
 **字段**  
 所模拟的方面的属性。  
  
 **“运算符”**  
 表达式运算符，例如， **=** 或 **Like**。  
  
 **预期值**  
 使属性表达式成功运行的字段值。  
  
 **实际值**  
 策略检测到的字段值。  
  
 **策略说明**  
 策略的说明。  
  
 **更多帮助**  
 单击超链接可打开与此策略有关的网页。 更多帮助超链接是在创建策略时配置的，可能为空或不可用。
