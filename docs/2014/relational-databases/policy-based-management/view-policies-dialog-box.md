# <a name="view-policies-dialog-box"></a>“查看策略”对话框
  此对话框用于查看对该目标有效的策略， 它显示相关的策略、其策略类别以及策略相关信息。  
  
## <a name="options"></a>选项  
 **策略运行状态**  
 指示与此目标有关的策略状态。 如果状态为未知，则保留空白。  
  
 **有效**  
 指示策略当前是否对目标有效。  
  
 **策略**  
 每个适用策略的名称。  
  
 **类别**  
 选定策略所属的策略类别。  
  
 **评估模式**  
 命名策略的评估模式。  
  
 **History**  
 单击超链接可显示执行历史记录报表。  
  
 **评估**  
 单击超链接可打开“评估策略”  对话框以执行策略。  
  
 **上次执行**  
 上次运行策略的日期和时间。
