# <a name="ipaddressname-property-servernetworkprotocolproperty-class"></a>IPAddressName 属性（ServerNetworkProtocolProperty 类）

## <a name="syntax"></a>语法  
  
```  
  
object  
.IPAddressName [= value]  
```  
  
## <a name="parts"></a>组成部分  
 *object*  
 一个表示实例上的网络协议属性的[ServerNetworkProtocolProperty 类](servernetworkprotocolproperty-class.md)对象 [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 。
