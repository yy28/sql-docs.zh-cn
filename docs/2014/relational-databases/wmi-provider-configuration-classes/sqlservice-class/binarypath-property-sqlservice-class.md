# <a name="binarypath-property-sqlservice-class-"></a>BinaryPath 属性（SqlService 类）
  获取服务的二进制路径。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.BinaryPath [= value]  
```  
  
## <a name="parts"></a>组成部分  
 *object*  
 一个表示服务的 [SqlService 类](sqlservice-class.md) 对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个指定服务的二进制路径的字符串值。
