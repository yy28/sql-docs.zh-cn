# <a name="name-property-securitycertificate-class"></a>Name 属性（SecurityCertificate 类）
  获取安全证书的名称。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.Name [= value]  
```  
  
## <a name="parts"></a>组成部分  
 *object*  
 一个表示安全证书的 [SecurityCertificate 类](securitycertificate-class.md) 对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个指定安全证书的名称的字符串值。
