## <a name="syntax"></a>语法  
  
```  
  
object  
.SHA [= value]  
```  
  
## <a name="parts"></a>组成部分  
 *object*  
 表示安全证书的 [SecurityCertificate Class] SecurityCertificate-class.md）对象。
