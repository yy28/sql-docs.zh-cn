# <a name="flagindex-property-clientsettingsgeneralflag-class"></a>FlagIndex 属性（ClientSettingsGeneralFlag 类）
  获取标志的索引值。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.FlagIndex [= value]  
```  
  
## <a name="parts"></a>组成部分  
 *object*  
 一个表示服务器设置的常规标志的 [ClientSettingsGeneralFlag 类](clientsettingsgeneralflag-class.md) 对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个指定标志索引值的 u`int32` 值。
