# <a name="flagvalue-property-clientsettingsgeneralflag-class"></a>FlagValue 属性（ClientSettingsGeneralFlag 类）
  获取标志的值。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.FlagValue [= value]  
```  
  
## <a name="parts"></a>组成部分  
 *object*  
 一个表示服务器设置的常规标志的 [ClientSettingsGeneralFlag 类](clientsettingsgeneralflag-class.md) 对象。
