# <a name="controlling-page-breaks-headings-columns-and-rows-report-builder-and-ssrs"></a>控制分页符、标题、列和行（报表生成器和 SSRS）
  分页符可将报表划分为单独的页，以便于查看和打印。 预览报表或将其导出为其他文件格式时，分页符可确定报表内容适合报表页的方式，以获得最佳查看效果。  
  
 添加分页符还会改进处理大型报表时的性能。 将显示呈现的页，同时在后台中呈现其余页。 这样，您就可以在等待查看报表的其他页面的同时，开始查看初始页。  
  
 分页符可以添加到表、矩阵、列表、图表、仪表或图像等报表项中。 还可以将分页符添加到表、矩阵或列表内的组中。 可以在组之前、组之后和组之间添加分页符。 默认情况下，不会向报表添加组与组之间的分页符。  
  
 有关详细信息，请参阅[在多个页中显示行标题和列标题（报表生成器和 SSRS）](../../reporting-services/report-design/display-row-and-column-headers-on-multiple-pages-report-builder-and-ssrs.md)和[在滚动报表时保持标题可见（报表生成器和 SSRS）](../../reporting-services/report-design/keep-headers-visible-when-scrolling-through-a-report-report-builder-and-ssrs.md)。
