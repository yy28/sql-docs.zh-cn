# <a name="create-new-condition-or-open-condition-dialog-box-description-page"></a>“创建新条件”或“打开条件”对话框，“说明”页
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  此对话框用于为基于策略的管理条件添加说明。  
  
## <a name="options"></a>选项  
 **Description**  
 键入条件的说明。  
  
 **创建日期**  
 创建条件的日期。  
  
 **创建者**  
 创建条件时所使用的登录名。  
  
 **修改日期**  
 上次更改条件的日期。  
  
 **修改者**  
 最近一次更改条件时使用的登录名。
